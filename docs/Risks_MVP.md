# Risks & Controls

## Legal

- Self Service must be positioned as space/tool usage, not mechanic labor by default.
- Shop Service terms must clearly state that the work is executed by the partner shop.

## Safety

- Mandatory agreement for self-service use
- Photo logging for self-service check-in
- Insurance handled in later phase

## Operational

- Reservation conflict enforced at DB level
- Package reservations must block time using 30-minute rounded duration
- Partner must publish package prices clearly
- Automatic settlement remains required where applicable

## Partner Risk

- Clear liability separation by reservation mode
- Partner is responsible for validating whether package work can finish within reserved time
- If the shop takes over because completion within reserved time is not feasible, full labor charge still applies
